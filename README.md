# Bit Valley Story · 比特谷物语

> A lab. Late night. One Enter key that should never have been pressed.

You wake up inside a machine that is slowly forgetting itself — and something deep in the data stream is calling your name. Explore the valley, solve its puzzles, and keep yourself referenced.

An open-world puzzle adventure built around the CAIE 9618 AS Computer Science syllabus. Every puzzle **is** a concept — logic gates, binary, assembly, SQL, circular queues, trace tables, phishing forensics and more — wrapped in a story about a machine, its residents, and the twenty years since anyone last logged in.

**Play it here → https://haoni0818.github.io/bit-valley-story/**

## How to play

- **WASD / arrow keys** move · **E** interact · **J** journal · **M** manual · **I** bag · **Q** scan · **Tab** map
- Default language is **English**; press the 中文 button on the title screen any time to switch (all technical terms stay in English).
- Stuck on a puzzle? Every puzzle has a 📖 concept card and a three-step hint ladder. Reading is never cheating.
- Works in any modern desktop browser (Chrome / Edge recommended). No install, no account.

## Saves

- Your progress saves automatically to this browser (localStorage).
- On shared computers, use **Esc → save export** to copy a save code, and import it on your own machine.
- Leaderboards and message boards are class features — pick a name you're happy for classmates to see.
